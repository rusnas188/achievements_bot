# 🚀 Achievements Bot

Telegram-бот для выдачи ачивок.

---

## 📦 Требования

Для работы бота необходимо установить:

- Docker
- Docker Compose 
- Docker Buildx

---

## ⚙️ Первичная настройка

### 1. Создание бота

Перед запуском нужно создать Telegram-бота через **@BotFather**.

1. Открой Telegram
2. Найди **@BotFather**  
3. Создай нового бота
4. Скопируй полученный токен

---

### 2. Получение ID главного администратора

Тебе потребуется Telegram ID главного администратора.

Получить его можно:
- через бота @userinfobot  
- либо через любого Telegram-бота, который показывает ID пользователя  

---

### 3. Конфигурация

Отредактируй файл `setting.conf`:

Замените значения:

- `YOUR_BOT_TOKEN` ➜ токен, полученный в пункте 1  
- `YOUR_TELEGRAM_ID` ➜ ID, полученный в пункте 2

```ini
bot_token = YOUR_BOT_TOKEN
admin_tg_id = YOUR_TELEGRAM_ID
```

---

## ▶️ Управление ботом

Управление осуществляется через скрипт `botctl`:

```bash
botctl start     # запуск бота
botctl stop      # остановка бота
botctl restart   # перезапуск бота
botctl status    # текущий статус
```
